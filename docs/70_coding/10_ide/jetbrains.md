# JetBrains

JetBrains is a company that makes IDEs for many languages. They are all very good and have a lot of features. They are also very customizable.

Each IDE has a free community edition and a paid ultimate edition. The community edition is enough for most people.

## [List of IDEs](https://www.jetbrains.com/all/)

| IDE           | Languages                                                                                              | Website                                                    |
| ------------- | ------------------------------------------------------------------------------------------------------ | ---------------------------------------------------------- |
| IntelliJ IDEA | Java, Kotlin, Groovy, Scala, JavaScript, TypeScript, SQL, HTML, CSS, PHP, Python, Ruby, Go, Rust, Dart | [JetBrains IntelliJ IDEA](https://www.jetbrains.com/idea/) |
| PyCharm       | Python                                                                                                 | [JetBrains PyCharm](https://www.jetbrains.com/pycharm/)    |
| WebStorm      | JavaScript, TypeScript, HTML, CSS                                                                      | [JetBrains WebStorm](https://www.jetbrains.com/webstorm/)  |
| CLion         | C, C++                                                                                                 | [JetBrains CLion](https://www.jetbrains.com/clion/)        |
